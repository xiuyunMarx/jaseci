- **Zero-config microservice deploys + ~2x faster cold k8s deploys**: the Kubernetes deploy now auto-discovers services and their public route prefixes from `sv import` (no `[plugins.scale.microservices.routes]` table needed; explicit routes still win), completing the config-less deploy story of #7137/#7140. Cold-deploy cost (#7138) drops roughly 2x end to end: the deploy driver ships its host-compiled `.jir` modules as re-homeable seeds that pods adopt before `jac install` (first boot validates instead of recompiling - rolling restarts benefit too), the default pod bootstrap no longer apt-installs packages or downloads bun (the shipped binary bundles both; declare extras via `[dependencies.system]`), and the e2e prints a per-phase timing report.
