- **Feature: Beta React Native target (`--client react-native`)**: Adds a second mobile path alongside Capacitor: `jac setup react-native`, `jac start --client react-native --dev`, and `jac build --client react-native` drive an Expo/Metro/Hermes scaffold under `.jac/mobile-rn/`. mobUI projects author UI in the `@jac/mobui` vocabulary (raw HTML tags are E1105 compile errors); for exempt `.cl.jac` web-boundary files the native runtime still maps familiar HTML tags (`div`, `button`, `input`, …) to React Native primitives, and `.native.cl.jac` files override web-only pieces when present. Includes a shared `JacClientCompiler` engine (the web `ViteCompiler` delegates to it), a native client runtime layered as platform adapters over `client_runtime_core`, dual-platform examples, and test coverage.
